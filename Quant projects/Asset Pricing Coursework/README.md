# Asset Pricing Coursework

This folder contains coursework notebooks in Python for discrete-time finance, continuous-time finance, option pricing, and supporting programming practice.

The notebooks are best read as a teaching sequence rather than as one single research project. Some are full programming sheets, while others are shorter worked examples.

## Recommended reading order

If you want the clearest path through the folder, use this order:

1. [00_intro_to_python_assignment.ipynb](./00_intro_to_python_assignment.ipynb)  
   Introductory Python and Jupyter workbook used to prepare for the finance coursework.

2. [01_discrete_time_finance_1.ipynb](./01_discrete_time_finance_1.ipynb)  
   First discrete-time finance programming sheet with binomial pricing foundations and core valuation exercises.

3. [02_discrete_time_finance_2.ipynb](./02_discrete_time_finance_2.ipynb)  
   Follow-on discrete-time notebook with wealth processes, constant-mix strategies, and limit behavior in binomial models.

4. [03_binomial_tree_examples.ipynb](./03_binomial_tree_examples.ipynb)  
   Shorter worked examples for binomial-tree pricing and replication logic.

5. [04_continuous_time_finance_1.ipynb](./04_continuous_time_finance_1.ipynb)  
   First continuous-time finance notebook covering Brownian motion, geometric Brownian motion, and simulation-based exercises.

6. [05_continuous_time_finance_2.ipynb](./05_continuous_time_finance_2.ipynb)  
   Applied option-pricing notebook covering Black-Scholes pricing, implied volatility, and volatility-surface work.

## Folder structure

- `00` is the programming prerequisite notebook.
- `01` to `03` are primarily discrete-time finance notebooks.
- `04` and `05` are the continuous-time and derivatives notebooks.

## Notes

- These notebooks were created as coursework and study material, so they preserve the exercise structure rather than forcing everything into one narrative style.
- The numbering and filenames were standardized afterward to make the folder easier to read from the outside.
