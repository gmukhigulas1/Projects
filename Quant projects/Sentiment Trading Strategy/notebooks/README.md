# Notebooks

This folder contains the Jupyter notebooks used for my Reddit sentiment trading research project.

The notebooks were written during different stages of the project, so they are not all equally polished. Some are used for data pulling and cleaning, while others focus on visualization, weighting, or strategy testing.

## Suggested reading order

If you are reviewing the project, I would read the notebooks in this order:

1. data collection / pulling notebook  
   Collects or prepares the raw Reddit and market-related data.

2. Reddit data processing notebook  
   Cleans Reddit posts/comments and prepares them for analysis.

3. ticker-mapping / combining notebook  
   Connects discussion data with company tickers or sector-level categories.

4. sentiment visualization notebook  
   Explores sentiment, engagement, and discussion activity over time.

5. weighting notebook  
   Tests different ways of weighting sentiment by engagement or other activity measures.

6. main model / strategy notebook  
   Tests the sentiment-based trading logic and compares results.

7. results visualization notebook  
   Produces charts for annual returns, cumulative returns, excess returns, and adjusted performance.
