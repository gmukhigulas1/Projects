# Stock-Market Analysis

Exploratory stock-market notebooks and small Python scripts.

Experimental work. I used these files to test ideas around price patterns, trendlines, simple backtesting, moving averages, sentiment strategy logic, and portfolio/Markowitz-style analysis.

## Files

- `stonks.ipynb` and `stonks.csv` — exploratory stock-data analysis
- `Sentiment Strategy.ipynb` — sentiment-related strategy testing
- `Markowitz bootstraping.ipynb` — Markowitz/portfolio-style analysis with bootstrapping ideas
- `HH-LL-HL-LH.ipynb` — higher-high, lower-low, higher-low, lower-high price-structure logic
- `TrendLineAutomation.py` — trendline automation script
- `highlow.py` — high/low price-structure script
- `smabacktest.py` — simple moving-average backtesting script
